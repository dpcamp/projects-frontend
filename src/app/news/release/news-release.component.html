New Updates

  • Authentication

    This application is now guarded against unauthorized users. If a user needs access please send an email to derek.campanile@vmsinc.org to have a user added

  • Individual Invoice and Requisition Management

    Invoices and Requisitions can now be accessed outside of their parent Job. This is useful in the event that an invoice has been assigned to the wrong job, or requisition assigned to the incorrect budget year. As a reminder, the only field editable in a requisition is the budget year.

  • Search

    You can now search for jobs, requisitions and invoices within each component. Jobs can be searched by Job Number, Invoices by Invoice Number and Requisition by Requisition Number.

  • "My Jobs" Feature

    Jobs that are assigned to you can be access using the "My Jobs" Link located under your name in the upper right, or from the Jobs component sub-menu.

  • Job Assignment dropdown

    You can now select the name of an authenticated user from the New Job, or Edit Job wizard. These jobs will instantly appear in the "My Jobs" section for that user.

Bug Fixes

  • First Time User Experience

    Fixed the issue where an authenticated user gets a blank page upon first time accessing the application.
